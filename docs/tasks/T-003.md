# T-003 Add Workflow and Adapter Capability Layer

## Goal

Add a portable workflow layer and adapter capability matrix so supported AI
coding tools know how to use Vibecoding Kit's prompt, policy, guard, and report
layers consistently.

## Background

v0.2.0 established the AI Agent Governance MVP: prompt modules, policy files,
command classification, risk reports, drift guard integration, and closeout
evidence. Generated projects still lack a concise workflow layer that explains
how agents should move through project scan, task creation, plan lock,
implementation, risk review, and closeout. Existing adapter bridge files explain
tool compatibility, but they do not yet point to reusable workflow modules or a
single capability matrix.
