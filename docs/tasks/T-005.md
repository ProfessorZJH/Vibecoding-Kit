# T-005 Governance Demos and UX Polish

## Goal

Add short, repeatable governance demos that show command classification, risk
reporting, and managed adapter block preservation alongside the existing AI
drift demo.

## Background

v0.4.0 added managed adapter blocks, giving the project a safe adapter update
primitive. The repository now has prompt, workflow, policy, guard, report, and
adapter maintenance layers, but a new user still needs several commands and
documents to see how those layers work.

T-005 turns the existing capabilities into 30-second demos. It does not add a
new CLI or change guard behavior. It adds demonstration scripts, expected
outputs, documentation links, and test-kit coverage so the demos stay honest.
