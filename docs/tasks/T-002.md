# T-002 Add Policy, Command Guard, and Risk Report

## Goal

Turn the current prompt-and-plan-drift baseline into a policy-backed AI agent
governance MVP with command classification and risk evidence.

## Background

The kit already has modular prompts, plan locking, drift checks, secrets
scanning, and closeout reporting. It still lacks a first-class policy layer, a
dedicated command guard, and a reusable risk report. This task adds those
pieces without changing the approved scope: documented policies in v1, a
classifier-only command guard, report-only risk classification, and closeout
evidence instead of hard blocking on HIGH or CRITICAL findings alone.
