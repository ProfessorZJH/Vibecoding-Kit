# v0.6.0 - Generated Project Smoke Demo

Release date: 2026-07-06

## Summary

v0.6.0 adds an executable generated project smoke demo. The release proves that
Vibecoding Kit can be installed into a clean target repository and that the
generated project can run its own preflight, drift/risk, and closeout scripts.

This release focuses on onboarding proof and demonstration quality, not new
guard or installer semantics.

## What's Included

- `examples/generated-project-demo/`: installs the kit into a temporary target
  project and runs the generated governance loop from inside that target.
- `examples/generated-project-demo/expected-output.txt`: records stable demo
  markers for installation, file checks, preflight, drift/risk, commit
  checkpoint, and closeout.
- `examples/README.md`: lists the generated project demo alongside the
  existing governance demos.
- README demo matrix with all five demo commands.
- `scripts/test-kit.sh` coverage for generated project demo presence,
  execution, and documentation references.

## Demo Commands

```bash
bash examples/ai-drift-demo/run-demo.sh
bash examples/command-risk-demo/run-demo.sh
bash examples/risk-report-demo/run-demo.sh
bash examples/adapter-block-demo/run-demo.sh
bash examples/generated-project-demo/run-demo.sh
```

## Generated Project Flow

The new demo verifies that a generated target project can run:

```bash
bash scripts/ai-preflight.sh T-000
bash scripts/drift-guard.sh
bash scripts/task-closeout.sh T-000 --no-tests --write-report
```

It also verifies that `reports/ai-risk/T-000-risk-report.md` and
`reports/ai-closeout/T-000.md` are written in the generated target project.

## Verification

Run from this kit repository:

```bash
git diff --check
bash -n examples/*/run-demo.sh
bash -n scripts/test-kit.sh
bash examples/generated-project-demo/run-demo.sh
bash scripts/test-kit.sh
```

## Known Limits

- No CLI is included in this release.
- No installer behavior changes are included.
- No YAML parser is added.
- No guard behavior changes are included.
- No automatic adapter sync is included.
- No demo depends on network access.
- Risk reports remain evidence for review; HIGH and CRITICAL risk are not
  standalone blockers in this release.
